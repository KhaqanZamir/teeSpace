## Overview

This repository contains the source code for a custom T-shirt printing website that allows users to create personalized designs and place print-ready orders. The platform provides an intuitive design editor, supports multiple product options, and streamlines the process from design creation to order submission.

The project is built with a focus on usability, scalability, and maintainable code structure, making it suitable for real-world e-commerce and print-on-demand workflows.
